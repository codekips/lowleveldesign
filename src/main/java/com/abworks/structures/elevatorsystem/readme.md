## Ask
design and implement elevator system that can handle multiple requests, move between floors, and 
manage direction and state.

## Requirements
**Multiple Elevators** : manage multiple  
**Request Handling**  : Will have requests to move UP or DOWN.  
**Manage Direction** : Maintains and updates its direction (UP, DOWN or IDLE)  
**States** : Elevator tracks current floor, direction and pending requests to it.  
**Move efficiently**: Service requests appropriately.  
**Extensibility**: Should be easy to add more elevators
